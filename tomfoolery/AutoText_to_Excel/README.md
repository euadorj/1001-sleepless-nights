## AutoTextExcel: Automated Data Structuring and Export Tool

### Project Background
This project happened because 
1. i wrote down a whole text list which had 60+ instances and i was lazy to do double work in posting to Excel again
2. preexisting solutions only copy pasted my text list into excel format, did not format.


This is a part of the various projects within "1001 sleepless nights" repo which i have yet to post. mainly various webapp using Python & C++, maybe a bit of Java

### Project Overview
Developed a console application utilizing C++ (& Python!) to automate the extraction and formatting of user-provided text data into structured CSV files suitable for Excel. 
This allows the user to convert their text list into a excel file for them to provide data.


### Technical Challenges
- Difficulty here was getting the regex function to work with the various different ways user could input
